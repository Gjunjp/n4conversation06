文法ぶんぽう 1

〜たほうがいいです

( 最好... / 建議做某事 )

相手あいてにアドバイスをする時（とき）に使（つか）います。

動詞（どうし）た形（けい） ＋ ほうがいいです
動詞（どうし）ない形（けい） ＋ ないほうがいいです
(用於給予對方建議或忠告，例如醫生建議病人「最好多休息」、「最好不要勉強」。)

アドバイスをしましょう！ (給予建議吧！)

「〜た/ない ほうがいいです」に変えてください。

1. 早はやく寝ねます。(早點睡覺)
➔ 熱ねつがあるから、早はやく ＿＿＿＿＿＿＿＿＿＿＿＿ 。

2. 無理むりをしません。(不要勉強)
➔ 今日は仕事しごとを休やすんで、無理むりを ＿＿＿＿＿＿＿＿＿＿＿＿ 。

アドバイスをしましょう！ 解答

「〜た/ない ほうがいいです」に変えてください。

文法ぶんぽう 2

〜なければなりません

( 必須... / 不做不行 )

ルールや義務ぎむを表あらわします。

会話（かいわ）では「〜ないといけません / 〜なきゃ」もよく使います。
(表示義務或規則，也就是「必須做某事」。在說明吃藥規定或看診手續時一定會聽到。)

動詞（どうし）ない形（けい）（ない を取る） ＋ なければなりません

例れい：病院びょういんで保険証ほけんしょうを出さなければなりません。

ルールを確認！ (確認規則吧！)

    ➔ 受付で保険証を                               

ルールを確認！ 解答

請將以下動作改為「必須...」的句型。

会話かいわ 1：診察しんさつを受うける (接受看診)

(你怎麼了嗎？)

(我從昨天開始發燒，而且喉嚨也很痛。)
お医者いしゃさん： 風邪かぜですね。薬くすりを出しておきますから、ゆっくり休やすんだほうがいいですよ。
(是感冒呢。我會開藥給你，你最好多休息喔。)
林： はい、わかりました。ありがとうございます。
(好的，我知道了。謝謝您。)
